# PesaSmart 

**From Knowledge to Wealth - The Kenyan Journey**

PesaSmart is a comprehensive fintech education platform designed specifically for the Kenyan market. It bridges the gap between financial literacy and real financial action through a three-phase journey: **Learn → Practice → Act**.

## Project Overview

### The Problem
- **76%** of Kenyans lack basic financial literacy (CBK Report 2023)
- Only **2.5%** insurance penetration (vs. 12% global average)
- Only **1 million** active NSE investors in a population of 50+ million
- Predatory digital lenders charging **15-30%** monthly interest

### The Solution
PesaSmart provides a guided three-phase journey:

| Phase | Name | Swahili | Purpose |
|-------|------|---------|---------|
| 1 | **LEARN** | "Pata Maarifa" | Financial education with Kenyan examples |
| 2 | **PRACTICE** | "Jaribu Bila Hasara" | Risk-free simulators with virtual money |
| 3 | **ACT** | "Fanya Biashara" | Real products with lowered barriers |

### Target Audience
- **Students (18-30):** No credit history, need school fees
- **Mama Mbogas:** Irregular income, need business capital
- **Jua Kali Artisans:** Need tool financing, materials purchasing
- **Smallholder Farmers:** Need crop insurance, input financing
- **Formal Employees:** Seeking side investments
- **Chamas/SACCOs:** Need group management tools
